# ssLOL API Client

A **universal Python client** for the [screenshare.lol (ssLOL) API](https://screenshare.lol), designed to work in **Discord bots, web apps, scripts, CLI tools, and other Python applications**.  
Supports **synchronous and asynchronous requests** with **typed responses** using Pydantic.

---

## **Installation**

### Using `pip`
```bash
pip install sslol-api


Requirements

Python 3.8+

Dependencies installed automatically: `requests`, `httpx`, `pydantic`

# sLOL API Client
1. API Key
Set your ssLOL API key in an environment variable:

#### Windows (PowerShell):
```$env:SSLOL_API_KEY="YOUR_API_KEY"```

### Linux/macOS (bash/zsh):
```export SSLOL_API_KEY="YOUR_API_KEY"```
